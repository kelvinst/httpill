# HTTPill

HTTP requests for sick people!

For more information about it, check our
[online docs](https://hexdocs.pm/httpill).

## Installation

The package can be installed by adding `httpill` to your list of dependencies
in `mix.exs`:

```elixir
def deps do
  [
    {:httpill, "~> 0.1.0"},
  ]
end
```
